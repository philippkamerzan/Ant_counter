# Объяснение программы про подсчёт муравьёв

Конечно. Представь, что у нас есть фотография, где много муравьёв, и мы
хотим, чтобы компьютер сам их нашёл и посчитал.

Этот файл --- программа на Python. Она работает как очень внимательный
помощник:

-   берёт картинку,
-   делает её удобнее для поиска,
-   ищет тёмные маленькие объекты,
-   пытается понять, где муравьи,
-   отделяет слипшихся муравьёв друг от друга,
-   считает их,
-   сохраняет картинки-подсказки, чтобы человек мог проверить результат.

Теперь объясню по-простому, как для 3 класса.

------------------------------------------------------------------------

## Главная идея

Программа говорит:

> "Дай мне фото. Я попробую найти на нём муравьёв и сказать, сколько
> их."

Но муравьёв искать не так просто, потому что:

-   на фото могут быть тени,
-   могут быть грязные пятнышки,
-   могут быть линии стенок,
-   муравьи могут слипаться в кучки.

Поэтому программа делает много маленьких шагов.

------------------------------------------------------------------------

## Что означают первые строки

``` python
#!/usr/bin/env python3
```

Это подсказка компьютеру: "Запускай этот файл как программу на Python
3."

``` python
import argparse
import os
import time
from datetime import datetime

import cv2
import numpy as np
```

Это как если бы программа сказала:

argparse --- мне нужен помощник, чтобы понимать команды при запуске;

os --- чтобы работать с папками и файлами;

time --- чтобы измерять время;

datetime --- чтобы писать текущую дату и время;

cv2 --- это OpenCV, очень важная библиотека для работы с картинками;

numpy --- библиотека для быстрых вычислений с числами и массивами.

------------------------------------------------------------------------

## Главная идея программы (простыми словами)

Эта программа: ищет на фотографии объекты, похожие на муравьёв,
разделяет их друг от друга и считает.
